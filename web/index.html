Emissivity (ε) is how efficiently a surface emits infrared compared to a perfect blackbody (ε=1.0). The Lepton assumes ε=1.0, so real materials — which emit less — read low. Lower ε with the slider and watch the corrected center value rise. Sensor accuracy is ±5°C; this is a teaching tool.
